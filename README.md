# 🚗 Smart Parking Optimization Website

## 📌 Overview
This project is a Smart Parking Optimization web application designed to help users find available parking spaces efficiently on campus. It provides real-time-like availability insights, supports search by parking lots, permits, and buildings, and suggests nearby alternatives to reduce congestion and save time.

The system is built with a modern frontend stack and focuses on usability, optimization, and clean UI design.

---

## ✨ Features
- 🔍 Search parking lots by **name, permit type, or nearby buildings**
- 📊 View **availability of parking spaces**
- 🗺️ Location-based filtering for better optimization
- 🚌 Displays **nearest Bull Runner stops and routes**
- 💡 Suggests alternative options when parking is limited
- 🎨 Clean UI built using **Figma-based components**

---

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Styling/UI: Figma-designed components
- Data: Structured parking lot dataset
- Tools: VS Code, Node.js, npm

## 🚀 How to Run the Project (Step-by-Step)

Follow these steps carefully to run the project on your local machine:

### 1. Extract the Project
- Download the ZIP file
- Extract it to a folder on your system

### 2. Open in VS Code
- Open **Visual Studio Code**
- Click on **File → Open Folder**
- Select the extracted project folder

### 3. Open Terminal in VS Code
- Click on **Terminal → New Terminal**

### 4. Install Dependencies
Run the following command:
```
npm install
```

### 5. Start the Development Server
Run:
```
npm run dev
```

### 6. Open in Browser
- After running the above command, you will see a local URL (usually something like):
```
http://localhost:5173/
```
- Open this URL in your browser to view the website
